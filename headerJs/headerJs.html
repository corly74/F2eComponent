3.利用 e.target 捕獲節點和 nodeName 節點來撰寫js
*使用 DOM .nodeName 屬性時兩條須遵守的規則 1. HTML 元素的節點名稱始終為大寫，即使它們是使用小寫字符顯式創建的。 將導致 a .nodeName === "HTML" 2. XML 元素的節點名稱始終採用原始大小寫，如創建時所指定的那樣。 將導致 a .nodeName === "data"， 將導致 a .nodeName === "DATA"。 因此創建時是用小寫，但取得 nodeName 的時候標籤會以大寫顯示
function checkList(e){
var num = e.target.dataset.num;
if(e.target.nodeName !== 'LI'){return};
alert('你選擇的農夫是'+country[num].farmer)
}
list.addEventListener('click',checkList,false);
